# Lab Steps

## Step 1 — Create an S3 Bucket
- Create a new S3 bucket in your AWS account
- Do not enable public access intentionally

## Step 2 — Review Public Access Block Settings
- Navigate to the bucket permissions tab
- Observe the four Public Access Block controls

## Step 3 — Enforce Public Access Block
- Enable all Public Access Block options
- Save changes

## Step 4 — Attempt Public Exposure
- Attempt to apply a public-read bucket policy
- Observe the failure or override prevention
