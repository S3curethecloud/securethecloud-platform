# STC Shield — Consulting Offers (Operator Outcomes)

## Offer 1 — Identity Risk Assessment (1–2 weeks)
**Goal:** identify identity trust abuse paths before attackers do.

Deliverables:
- Shield attack-path report (exec + technical)
- Visual identity graph evidence
- Risk-ranked remediation roadmap

## Offer 2 — Zero Trust Identity Review
Focus areas:
- SSO & federation trust boundaries
- Cross-account / cross-project access
- Permission set design + least privilege

Deliverables:
- Trust boundary mapping
- Risk findings tied to abuse paths
- Control recommendations and validation steps

## Offer 3 — Cloud Breach Readiness
Core question:
“If an identity is compromised today, what happens next?”

Deliverables:
- “blast radius” identity-path model
- top 5 critical choke points
- detection + hardening plan
