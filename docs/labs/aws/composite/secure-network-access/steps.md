# Lab Steps

## Step 1 — Define Network Boundaries
- Identify public vs private subnets
- Restrict ingress points

## Step 2 — Apply Security Group Controls
- Allow only required ports and sources
- Deny broad CIDR access

## Step 3 — Enforce Access Path
- Access internal resources via approved entry point
- Block direct public access

## Step 4 — Observe Traffic
- Review VPC Flow Logs or access attempts
