# AWS KMS Basics

## Problem
Sensitive data is often stored or transmitted without strong encryption
or proper key management, increasing breach impact.

## Why It Matters
Encryption at rest and in transit is a baseline control for regulated
and enterprise environments. KMS centralizes key management and auditing.

## What You Will Learn
- How AWS KMS manages customer master keys
- How services integrate with KMS
- How to audit key usage

## Real-World Context
KMS is foundational for protecting S3, EBS, RDS, and secrets across AWS.
