# Lab Steps

## Step 1 — Review Default CloudTrail
- Navigate to the CloudTrail console
- Observe the default trail configuration

## Step 2 — Create or Validate a Trail
- Ensure management events are enabled
- Configure S3 as the log destination

## Step 3 — Enable Multi-Region Logging
- Enable logging across all regions
- Confirm future regions are included

## Step 4 — Generate Activity
- Perform IAM or S3 actions
- Allow time for logs to be delivered
