# AWS S3 Public Access Block

## Problem
Amazon S3 is frequently involved in public data exposure incidents due to
misconfigured bucket policies, ACLs, or disabled Public Access Block settings.

## Why It Matters
Publicly exposed storage can lead to data breaches, regulatory violations,
and reputational damage. S3 Public Access Block provides a centralized control
to prevent accidental exposure.

## What You Will Learn
- How S3 Public Access Block works at the bucket and account level
- How it interacts with bucket policies and ACLs
- How to verify enforcement correctly

## Real-World Context
This control is foundational in healthcare, finance, and enterprise AWS
environments and is commonly assessed during security audits.
